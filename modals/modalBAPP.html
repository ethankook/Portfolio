BAPP: A Simple Budgeting App

In Progress...

Inspiration: Managing personal finances can be overwhelming, especially for students and young professionals. We wanted to build a simple, intuitive tool that helps users visualize their spending and build better money habits—no spreadsheets or bank account connections required.

What it does: BAPP is a lightweight budgeting app where users can set a total budget and allocate spending across custom categories (e.g., groceries, rent, entertainment). Through a clean dashboard and modal-based editing system, users can easily track their expenses and remaining funds in each category.

How we built it: The backend is built with Spring Boot and uses PostgreSQL to store budget data. The frontend is developed with React and TypeScript for a responsive, accessible user experience. We prioritized usability and left out account integration to keep the app lean and immediately functional.

Challenges: We faced challenges designing a modal system that felt smooth and non-intrusive while ensuring backend data remained consistent. Synchronizing real-time data between components also required thoughtful state management and API handling.

Accomplishments: We're proud of delivering a full-stack app with clean design, smooth interactions, and modular code. It’s something we actually find useful ourselves, and we kept the codebase organized for future growth.

What we learned: We gained experience building full-stack applications from scratch, integrating React with Spring Boot, and designing maintainable state and modal flows. We also learned how to convert user needs into clear, actionable features within a tight development timeline.

What’s next: We plan to add persistent sessions, recurring budget support, visualizations (like pie charts and progress bars), CSV export, mobile responsiveness, and secure authentication for personalized tracking.

Built With: Java, Spring Boot, React, TypeScript, PostgreSQL
