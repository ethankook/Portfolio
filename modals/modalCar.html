Autonomous Racing Robot

Completed

Github Repository

Overview: A fully autonomous robot built to navigate tracks of varying complexity using analog photoresistor input and PID feedback control. The robot was tuned for performance across multiple competition scenarios including loops, drag races, and obstacle-heavy tracks.

Key Features:

  • Dynamic PID tuning through potentiometers
  • Adjustable base speed for terrain/track specific performance
  • LED underglow for sensor stabilization and visibility
  • 3D modeled chassis(Onshape) and custom-soldered protoboards

Technology Used: Arduino, C++, analog sensors, soldering tools, motor drivers, signal debugging with multimeters and oscilloscopes

Process: Initial builds focused on chassis stability and signal fidelity from sensors. Iterative testing led to multiple board redesigns to improve light detection accuracy. PID parameters were fine-tuned based on track types. LEDs were added to ensure consistent lighting. Components were secured with hot glue to prevent disconnection under stress.

Challenges: Sudden signal loss due to inconsistent lighting, battery voltage drops, and PID instability on sharp curves. Debugging required real-time observation and repeated test loops with adjusted parameters.
