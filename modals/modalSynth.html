Digital Audio Synthesizer

Completed Project

Github Repository

Overview: A powerful digital audio synthesizer developed in MATLAB, offering users a hands-on interface to explore sound generation, manipulation, and effects processing. The app allows real-time waveform generation, audio transformation, and visualization within an intuitive GUI.

Key Features:

  • Generation of sine, square, sawtooth, triangle, and ECG waveforms
  • Audio effects: echo, reverb, chorus, flanger, distortion, delay, muffled
  • Volume, pitch, and speed controls
  • Real-time FFT visualization and signal playback
  • Envelope shaping and white noise synthesis
  • Interactive GUI for all controls using MATLAB App Designer

Technology Used: MATLAB, MATLAB App Designer, Audio Toolbox

Process: The project began with core waveform functions and expanded into a full interface. Each audio effect was modularly designed and chained using internal signal flow logic. Visual elements such as spectrum analyzers and time plots were created for user interaction and feedback.

Challenges: Handling overlapping effects without signal clipping, managing GUI state transitions, and ensuring responsiveness in real-time audio processing.

Accomplishments: A polished and flexible synthesis tool capable of both educational use and creative sound design. Achieved a robust real-time interface with interactive controls and audio output.

Applications: DSP education, sound design experimentation, signal chain prototyping, and audio effect testing.
